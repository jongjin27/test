RPG Battle Game

이 프로젝트는 Dart로 제작된 간단한 텍스트 기반 RPG 전투 게임입니다. 플레이러는 캐릭터를 생성하고 다양한 몬스터들과 전투를 벌이며 생존을 목표로 합니다.

기능
1.캐릭터 생성
2. 몬스터와 전투
3. 게임 저장
4. 몬스터 데이터 불러오기
5. 캐릭터 데이터 불러오기

파일 설명
1. main.dart : 게임의 주요 실행 파일
2. characters.txt  : 캐릭터 정보 저장
3. monsters.txt    : 몬스터 정보 저장
4. result.txt      :게임 결과 정보 저장
